


# Проект магистерской диссертации 
---

## ✨ Возможности

*   **Выбор фрейма**: Автоматическое определение текущего выделения.
*   **Анализ структуры**: Получение списка всех дочерних элементов (имя, тип).
*   **Доступ к Variables**: Проверка привязанных дизайн-переменных к заливке (fills).
*   **CSS-инспектор**: Получение актуальных CSS-свойств выбранного фрейма (как в панели Inspect).
*   **Экспорт**: Быстрое сохранение фрейма в формате PNG в масштабе 2x.
*   **Интерфейс**: Удобное окно плагина с выводом логов в формате JSON.

## 🛠 Технологический стек

*   **TypeScript**: Логика плагина (API Figma).
*   **HTML/JS**: Интерфейс пользователя (UI).
*   **Figma API**: Взаимодействие с объектами на холсте.

## 📁 Структура файлов

*   `code.ts` — Основная логика плагина (работа с API Figma).
*   `ui.html` — Интерфейс плагина (кнопки, стили и скрипты скачивания).
*   `manifest.json` — Конфигурация плагина и разрешения.
*   `code.js` — Скомпилированный файл (генерируется автоматически).

## 📝 Использование

1. Выделите **Frame** на холсте Figma.
2. Запустите плагин через меню `Plugins -> Development`.
3. Нажмите кнопку **"Анализировать выбранное"**, чтобы увидеть данные в формате JSON.
4. Нажмите **"Экспорт в PNG"**, чтобы мгновенно скачать изображение фрейма.
